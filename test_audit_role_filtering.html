Available Endpoints

This test demonstrates the new role-specific audit log filtering endpoints:

  • GET /api/admin/audit-logs/admin - Filter logs by admin users only
  • GET /api/admin/audit-logs/teacher - Filter logs by teacher users only
  • GET /api/admin/audit-logs/student - Filter logs by student users only
  • GET /api/admin/audit-logs - General endpoint with user_role parameter
