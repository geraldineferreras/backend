Debugging Information

Common Issues:

  • File not being sent: Check if your frontend is using FormData correctly
  • Wrong field name: The API expects 'attachment' as the field name
  • Content-Type header: Must be 'multipart/form-data' for file uploads
  • File size: Maximum 10MB allowed
  • File type: Only specific file types are allowed

Expected Request Format:


// Multipart Form Data
Content-Type: multipart/form-data
Authorization: Bearer your_jwt_token

Form Fields:
- submission_content: "Your text content"
- class_code: "MATH101"
- attachment: [file data]

Frontend Code Example:


const formData = new FormData();
formData.append('submission_content', 'Your content');
formData.append('class_code', 'MATH101');
formData.append('attachment', fileInput.files[0]);

fetch('/api/tasks/23/submit', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer ' + token
  },
  body: formData
});
