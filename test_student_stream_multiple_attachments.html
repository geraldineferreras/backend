Student Stream API Test

This test will retrieve stream posts from a classroom to see if multiple attachments are properly displayed.

Instructions

  1. First, create a teacher post with multiple attachments:
    • Use the teacher test file to create a post with multiple files
    • Make sure the post is published (not draft)
  2. Then test the student view:
    • Get a valid student JWT token
    • Enter the class code and token above
    • Click "Get Stream Posts" to see if multiple attachments appear
  3. Expected result:
    • Posts with multiple attachments should show an "attachments" array
    • Each attachment should have file details and serving URLs
    • Backward compatibility should be maintained for single attachments
